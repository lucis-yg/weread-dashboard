# 📚 WeRead Dashboard

一个优雅的微信读书数据可视化工具，基于 Vue 3 开发，提供书架管理、笔记查看、阅读统计等功能。()

![首页](./images/shouye.png)

## ⚠️ 重要提醒：需要配合后端服务使用

本项目是**前端展示层**，必须配合后端 API 服务才能正常工作。

- **后端项目地址**：[weread-skill-api](https://github.com/lucis-yg/weread-skill-api)
- **启动顺序**：请先启动后端服务，再启动本项目
- **API 配置**：默认请求 `/api/v1`，如后端地址不同，请修改 `src/api/weread.js` 中的 `baseURL`


## ✨ 功能特性

| 模块 | 功能 |
|------|------|
| 📖 首页 | 阅读概况统计、最近阅读展示 |
| 📚 书架 | 书籍列表、读完/在读筛选 |
| ✏️ 笔记 | 笔记汇总、划线内容、个人想法 |
| 📊 统计 | 阅读天数、时长、偏好分析 |
| 🔍 书籍详情 | 基本信息、阅读进度、章节目录、热门划线、书友点评、相似推荐 |

## 🛠️ 技术栈

- **框架**：Vue 3 + Vite
- **路由**：Vue Router 4
- **样式**：Tailwind CSS
- **HTTP**：Axios
- **图标**：Emoji

## 📦 快速开始

### 环境要求

- Node.js 22+
- npm 或 pnpm

### 安装依赖

```bash
npm install
# 或
pnpm install
```

## 📝 已知不足

### 功能未完全覆盖

本项目已实现大部分常用功能，但**并非所有后端 API 都有对应的前端页面**，例如搜索功能尚未添加前端界面。

### 后续计划

本项目最初目的是**验证后端微信读书 API 的可用性**，因此前端仅实现了核心的数据展示功能。考虑到实际使用场景和投入产出比，**暂不计划补充缺失的增删改功能**。

如果你需要完整的读写操作，欢迎基于本项目二次开发或提交 PR。
